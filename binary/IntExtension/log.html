log method

int log(
  1. [int? base]
)

Returns the natural logarithm of this integer, rounded down.

this must be a positive integer.

If base is provided, the logarithm is calculated with respect to that base; for example, log(10) returns the base 10 logarithm of this, and base must be a positive integer greater than 1.

Example

10.log(); // 2
10.log(2); // 3
10.log(10); // 1

Implementation

@pragma('dart2js:tryInline')
@pragma('vm:prefer-inline')
int log([int? base]) {
  final a = math.log(this);
  return switch (base) {
    null => a.toInt(),
    2 => a ~/ math.ln2,
    10 => a ~/ math.ln10,
    1 => throw ArgumentError.value(base, 'base', 'must be greater than 1'),
    _ => a ~/ math.log(base),
  };
}
